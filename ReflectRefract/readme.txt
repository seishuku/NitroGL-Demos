Reflection And Refraction

Keys:
Left Mouse Button + Drag = Rotate
Right Mouse Button + Drag = Zoom
Keypad '+' = Bump Scale Up
Keypad '-' = Bump Scale Down
M = Change Model
T = Change Textures (Base and Bump)
