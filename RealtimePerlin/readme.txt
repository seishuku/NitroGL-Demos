Realtime Perlin Noise, with color transfer provided by a Fragment Shader.

Controls:
Left mouse button + Drag = Rotate Quad
Right mouse button + Drag = Zoom Quad
Up/Down arrows = Increase/Decrease number of iterations
T = Enable/Disable color transfer
Spacebar = Pause animation
0-9 = Select different transfer maps (1 being the first, and 0 being the last)
