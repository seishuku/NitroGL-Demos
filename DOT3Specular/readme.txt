Per pixel DOT3 bump mapping with per pixel specular highlights

Controls:
Left mouse button + Drag = Rotate
Right mouse button + Drag = Zoom
Left mouse button + Control + Drag = Move light
Right mouse button + Control + Drag = Zoom light
W = Wireframe (might be slow with some models)
T = Load different textures
M = Load different model
A/Z = Red light color +/-
S/X = Green light color +/-
D/C = Blue light color +/-
Escape = Quit

Known issue:
When loading a texture or model and the open dialog is over the main window,
and if you double click the file name, it will mess up the model rotations
(this can be fixed by moving the open dialog away from the main window).
