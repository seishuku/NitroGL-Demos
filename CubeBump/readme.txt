Cubic Enviroment Bump Mapping

Warning:  May cause lock-ups for unknown reasons (probably has something to do with the drivers)

Controls:
Left mouse button + Drag = Rotate
Right mouse button + Drag = Zoom
T = Load different textures
M = Load different model
L = Show vectors
P = Show wireframe (may be slow)
Numpad + = Bump scale increase (max of 2.0)
Numpad - = Bump scale decrease (min of 0.0)
